# GosShooter

Demo 2D em Python com Pygame.

## Como executar

1. Instale o Python 3.11+.
2. No terminal, entre na pasta do projeto.
3. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```
4. Rode o jogo:
   ```bash
   python main.py
   ```

## Controles

- **WASD** ou **Setas**: mover
- **Mouse**: mirar
- **Clique esquerdo**: atirar
- **ENTER**: iniciar no menu
- **R**: reiniciar após derrota
- **ESC**: voltar ao menu

## Estrutura

- `main.py`: ponto de entrada
- `code/`: código separado em módulos
- `assets/images/`: sprites e telas
- `assets/sounds/`: efeitos sonoros
